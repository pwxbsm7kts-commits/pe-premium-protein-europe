# pe-premium-protein-europe
This project simulates a private equity origination mandate focused on identifying acquisition targets among European mid-market premium red meat and branded protein producers.

The objective is to:

1. Validate the structural demand shift toward premium and health-oriented protein consumption

2. Map and screen the European mid-market protein universe

3. Identify platform and add-on acquisition targets

4. Develop a systematic prioritisation framework

5. Support the thesis using firm-level and macroeconomic data

Tools: Orbis, CapIQ, Excel, Stata
